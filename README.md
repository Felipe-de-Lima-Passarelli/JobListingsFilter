# 💼 Job Listings Filter

Uma aplicação Front-End desenvolvida com **Next.js** e **TypeScript** que permite filtrar vagas de emprego dinamicamente com base em múltiplas tags selecionadas pelo usuário.

---

## 🚀 Descrição

O **Job Listings Filter** é uma interface interativa onde o usuário pode visualizar uma lista de vagas e aplicar filtros combinados para refinar os resultados.

A lógica de filtragem foi construída para garantir precisão: apenas vagas que atendem **todos os critérios selecionados** são exibidas.

---

## 🛠 Tecnologias utilizadas

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Next Image Optimization

---

## ⚙️ Funcionalidades

- 📋 Listagem de vagas
- 🏷️ Filtro por múltiplas tags
- ❌ Remoção individual de filtros
- 🧹 Limpar todos os filtros
- ⚡ Atualização dinâmica da lista
- 🎨 Interface responsiva
